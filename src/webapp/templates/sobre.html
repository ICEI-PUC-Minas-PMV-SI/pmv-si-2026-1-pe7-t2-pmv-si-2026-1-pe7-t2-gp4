Entrega da Etapa 5

Esta interface foi desenvolvida de forma isolada em src/webapp/ para cumprir o requisito de aplicação em produção com inferência dinâmica, sem alterar notebooks ou relatórios anteriores.

Tecnologias utilizadas

Python FastAPI scikit-learn Pydantic Jinja2 HTML / CSS / JS

Arquitetura resumida

Monólito Python com API REST e páginas server-side. O modelo é carregado uma única vez a partir de artifacts/no_show_model.joblib. Frontend e backend compartilham o mesmo projeto, sem banco de dados e sem persistência de formulários.
